🚀 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. YOU UPLOAD A PDF
   Click paperclip → Select PDF file

2. BROWSER EXTRACTS TEXT
   Uses PDF.js (industry standard)
   All processing happens on YOUR device
   Shows "…" → "✓" indicator as it works

3. TEXT GETS MERGED WITH YOUR MESSAGE
   Your question + PDF content combined
   Example: "Résumé this PDF"
            + [Full PDF text]
            = Enriched message to AI

4. AI UNDERSTANDS BOTH
   Your question + The document
   Returns intelligent, contextual answers

5. CONVERSATION STAYS CLEAR
   File badges show what you sent
   "You sent: lesson.pdf, question.jpg"
   Chat history stays organized


💡 INTELLIGENT DESIGN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Why This Approach Is Smart:

  ✓ CLIENT-SIDE EXTRACTION
    No server processing needed
    Instant feedback to user
    Zero latency for text extraction

  ✓ SIZE OPTIMIZED
    Max 8,000 chars per PDF (~20 pages)
    Prevents token overflow to AI
    Still handles most documents perfectly

  ✓ MULTIPLE FILES SUPPORTED
    Upload 1, 2, 3+ PDFs in one message
    All texts combined intelligently
    Clear formatting for AI understanding

  ✓ CACHE SYSTEM
    Extracted text cached during session
    Remove + re-add same file = instant
    Clear cache on new conversation

  ✓ CLEAR STRUCTURE
    PDF content marked: "📄 **Contenu du PDF: name.pdf**"
    AI knows exactly what's PDF vs. question
    Natural language understanding enhanced

📊 TECHNICAL ARCHITECTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Frontend Flow:
  [User selects file]
       ↓
  [Detect: Image or PDF?]
       ↓
  IF IMAGE: Show thumbnail
  IF PDF: Extract text asynchronously
       ↓
  [Cache extracted text]
       ↓
  [Show status: "…" → "✓"]
       ↓
  [User sends message]
       ↓
  [Enrich message with PDF text]
       ↓
  [Send enriched message + display files]


Backend Flow:
  [Receive enriched message]
       ↓
  [Pass to Gemini/Groq API]
       ↓
  [AI reads both question + PDF]
       ↓
  [Generate intelligent response]
       ↓
  [Send back to user]

  🔑 KEY: No changes to backend!
     It receives the full enriched text
     Already handles it perfectly
